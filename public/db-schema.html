PRODOTTO
belongsTo ────────────────
CATEGORIA
N : 1 — Un prodotto appartiene a una categoria
FK: productos.categoria_id → categorias.id
VARIANTE_PRODOTTO
belongsTo ────────────────
PRODOTTO
N : 1 — Più varianti per prodotto (as: "variantes")
FK: variantes_producto.producto_id → productos.id
IMMAGINE_PRODOTTO
belongsTo ────────────────
PRODOTTO
N : 1 — Galleria immagini (as: "imagenes")
FK: imagenes_producto.producto_id → productos.id
RECENSIONE
belongsTo ────────────────
PRODOTTO + UTENTE
N : 1 — Recensione su prodotto, opzionale utente
FK: resenas.producto_id → productos.id
FK: resenas.usuario_id → usuarios.id (null ok)
ORDINE
belongsTo ────────────────
UTENTE
N : 1 — Ordine collegato a utente (guest ok)
FK: ordenes.usuario_id → usuarios.id (null ok)
DETTAGLIO_ORDINE
belongsTo ────────────────
ORDINE + PRODOTTO
N : 1 — Riga ordine con prodotto e quantità (as: "detalles")
FK: detalles_orden.orden_id → ordenes.id
FK: detalles_orden.producto_id → productos.id
ARTICOLO
belongsTo ────────────────
UTENTE
N : 1 — Articolo blog scritto da admin
FK: articoli.autore_id → usuarios.id
TICKET
belongsTo ────────────────
UTENTE
N : 1 — Ticket di supporto per utente
FK: tickets.usuario_id → usuarios.id
MESSAGGIO
belongsTo ────────────────
TICKET
N : 1 — Messaggi chat nel ticket (as: "messaggi")
FK: mensajes.ticket_id → tickets.id
VISITA
belongsTo ────────────────
PRODOTTO
N : 1 — Visita pagina prodotto (nullable)
FK: visitas.producto_id → productos.id (null ok)
